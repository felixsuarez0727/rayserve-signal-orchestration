# UI – Diagrama de la Red Multitarea

Interfaz ligera en React (por CDN) para visualizar la arquitectura: extractor compartido + cabezas de clasificación y SNR.

## Cómo abrir

1. Abre `ui/index.html` en el navegador, o sirve la carpeta:
```
cd ui
python -m http.server 8080
# abre http://localhost:8080
```

No requiere instalación (usa React/ReactDOM por CDN).

## Personalización
- Edita capas en `ui/index.html` (componentes `Card`).
- Ajusta estilos en `ui/styles.css`.
